🚀 Built on Web4

The ecosystem that powers the intelligent, user-owned Web4 experience:

  • 🪙 fadakacoin
    Native cryptocurrency shell for the Fadaka Blockchain—includes wallet integration (MetaMask/Trust Wallet), analytics (RODAAI), and the Lola virtual assistant.
  • 🌐 fadaka-blockchain
    Core Go-based chain: transaction routing, token logic, peer networking, and consensus.
  • 💸 fadakatoken
    ERC20-style smart contract implementation for FDAK tokens—supports minting, burning, and upgrades.
  • 🧠 project_pilot_ai
    GPT-powered assistant designed to orchestrate blockchain workflows, automate code, and support developer queries.
  • 🤖 congen
    Engine for chatbots assistant trained on structured data—streamlined for Streamlit deployments.
  • 🧪 UniversalCertForge
    FastAPI-powered certificate tool for SSL, app, and binary signing—helps secure Web4 infrastructure.
  • 💬 zetachat
    Real-time AI-enabled chat and collaboration built with FastAPI and WebSockets.
  • 🔌 web4walletconnect
    JavaScript library designed to connect wallets across Web4 apps—mobile and desktop compatible.
  • 🌐 web4wallet (UI)
    The Swift Beta wallet interface—integrated token swaps, QR code handling, AI dashboard, and more, ready for GitHub Pages.
