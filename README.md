## 🚀 Getting Started

1. **Clone this repo:**
```bash
git clone https://github.com/Web4application/web4-node.git
cd web4-node
```

2. **Run everything:**
```bash
docker-compose up --build
```

3. **Access interfaces:**
- Wallet UI: `http://localhost:3000`
- AI API: `http://localhost:8000`
- Fadaka node: `localhost:8545`
- Docs: `/docs` folder or GitHub Pages
